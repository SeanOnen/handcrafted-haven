# Handcrafted Haven

A full-stack marketplace web application where artisans and crafters can showcase and sell their unique handcrafted items. Handcrafted Haven connects talented creators with customers who appreciate the beauty and quality of handmade products.

## Features

- **Seller Profiles** – Authenticated sellers can showcase their craftsmanship, share their story, and display their handcrafted items.
- **Product Listings** – Artisans can list items with descriptions, pricing, and images. Users can browse and filter by category or price.
- **Reviews & Ratings** – Any user can leave a rating and written review on a product.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/handcrafted-haven.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
